# Changelog

## 0.2.0 — Portál a automatizace

- nový sci-fi design portálu inspirovaný hvězdnou bránou,
- animovaný centrální portál a hvězdné pozadí,
- řídicí centrum Automatizace,
- manifesty aplikací a automatická synchronizace verzí,
- hodinové GitHub Actions a volitelný okamžitý repository dispatch,
- připravenost pro přidávání dalších aplikací,
- aktualizace registru na Generátor 7.0.4, Diferenciátor 1.0.1, LUDUS 1.14.1 a Korespondenčního asistenta 4.0.1,
- nový manifestový JSON kontrakt a sync report,
- přepracovaný tmavý vizuální systém všech stránek.

## 0.1.0 — První serverless portál

- společný rozcestník čtyř aplikací,
- demo, knihovna, bezpečnostní centrum a pilotní přehled,
- PWA a automatické nasazení přes GitHub Pages.
