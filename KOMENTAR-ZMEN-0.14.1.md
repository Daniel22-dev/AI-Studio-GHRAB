# AI Studio GHRAB 0.14.1 – oprava importu evidence

Tato opravná verze nahrazuje nespolehlivé programové otevírání skrytého výběru souboru nativním ovládáním. Import nyní funguje stejně v běžném prohlížeči i v nainstalované PWA.

## Změny

- jedno tlačítko načte přístupový soubor i zálohu evidence,
- soubor lze také přetáhnout do vyznačené plochy,
- Studio automaticky rozpozná typ souboru,
- po importu zobrazí přesný počet načtených záznamů nebo konkrétní chybu,
- přidán soukromý jednorázový import přes fragment adresy; fragment se po načtení okamžitě odstraní z adresního řádku.

Osobní údaje kolegů nejsou vloženy do veřejného zdrojového kódu ani do veřejného GitHub repozitáře.
