# Nahrání AI Studia GHRAB 0.11.0

1. Rozbalte ZIP.
2. Nahrajte jeho obsah přímo do kořene repozitáře AI Studio GHRAB.
3. Nechte přepsat původní soubory včetně skryté složky `.github`.
4. Proveďte commit a počkejte na zelené GitHub Actions.
5. Otevřete AI Studio, proveďte `Ctrl + F5` a poté znovu otevřete dílčí aplikace.

Dílčí aplikace není nutné kvůli samotnému hlášení chyb znovu upravovat: jejich chráněné stránky načítají centrální `app-guard.js` z AI Studia. Pokud však ještě nebylo nasazeno vydání 0.10.0, nahrajte nejdříve jeho balíky aplikací a AI Studio 0.11.0 jako poslední.
