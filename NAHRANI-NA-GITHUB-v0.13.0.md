# Nahrání AI Studio GHRAB 0.13.0 na GitHub

1. Otevřete repozitář **AI-Studio-GHRAB**.
2. Nahrajte **obsah složky `AI-Studio-GHRAB-main`**, nikoli samotnou nadřazenou složku.
3. Potvrďte přepsání existujících souborů.
4. Proveďte commit, například `AI Studio 0.13.0 – mechanické prstence brány`.
5. Počkejte, až GitHub Actions zezelená.
6. Otevřete nasazené AI Studio a proveďte `Ctrl + F5`. U nainstalované PWA ji úplně zavřete a znovu spusťte.
7. Klikněte na některou odemčenou aplikaci a ověřte tři pohyblivé prstence, sedm zámků a postupné stavové hlášky.

Není nutné před tímto balíkem nahrávat novou verzi dílčích aplikací; změna se týká pouze AI Studia.
