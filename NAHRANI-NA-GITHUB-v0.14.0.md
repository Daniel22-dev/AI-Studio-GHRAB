# Nahrání AI Studia GHRAB 0.14.0 na GitHub

1. Rozbalte balíček.
2. Otevřete složku `AI-Studio-GHRAB-main`.
3. Nahrajte její obsah do kořene repozitáře AI Studia a potvrďte nahrazení existujících souborů.
4. V GitHub Desktopu vytvořte commit, například `AI Studio 0.14.0 – evidence přístupů`.
5. Klikněte na **Push origin**.
6. Počkejte, až GitHub Actions skončí zeleně.
7. Ve Studiu otevřete **Správa → Evidence přístupů**.
8. Starší přístup kolegyně načtěte přes **Importovat přístupový soubor**, případně použijte dodanou soukromou zálohu evidence.

Při prvním otevření po aktualizaci doporučuji jednou obnovit stránku pomocí `Ctrl + F5`.
