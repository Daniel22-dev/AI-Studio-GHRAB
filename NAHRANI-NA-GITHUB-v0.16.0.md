# Nahrání AI Studio GHRAB 0.16.0 na GitHub

1. Stáhněte a rozbalte balíček `AI-Studio-GHRAB-v0.16.0-GITHUB-READY.zip`.
2. Otevřete repozitář AI Studio GHRAB na GitHubu.
3. Zvolte **Add file → Upload files**.
4. Nahrajte celý obsah rozbaleného balíčku přímo do kořene repozitáře.
5. Potvrďte přepsání existujících souborů.
6. Jako zprávu commitu lze použít:

   `v0.16.0 – filmové spuštění, holografické panely a režim projekce`

7. Počkejte, až GitHub Actions dokončí nasazení zeleným stavem.
8. Otevřete AI Studio a proveďte `Ctrl + F5`.
9. U nainstalované PWA ji úplně ukončete a znovu otevřete.

## Rychlá kontrola po nasazení

- domovská stránka se načte bez chyby,
- na PC jsou kolem brány vidět jemné holografické panely,
- kliknutí na povolenou aplikaci spustí filmovou sekvenci,
- tlačítko Přeskočit animaci otevře aplikaci okamžitě,
- stránka Prezentace nabízí tlačítko Spustit projekci,
- Poznámky v novém okně se otevřou odděleně,
- Správa, Evidence a ostatní pracovní stránky zůstávají bez nových 3D efektů.

## Návrat na původní verzi
Při potřebě návratu znovu nahrajte obsah uloženého původního balíčku 0.14.4 a proveďte `Ctrl + F5`.
