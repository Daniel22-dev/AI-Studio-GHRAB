# Nahrání AI Studio GHRAB 0.6.0 na GitHub

1. Rozbalte pouze balík `AI-Studio-GHRAB-v0.6.0-GitHub.zip`.
2. Nahrajte jeho obsah do kořene repozitáře AI Studio GHRAB a potvrďte přepsání starších souborů.
3. Soukromý administrátorský ZIP **nikdy nenahrávejte na GitHub, Drive sdílený s kolegy ani školní web**.
4. Po dokončení GitHub Actions otevřete Studio v anonymním okně. Všechny aplikace mají být viditelné, ale uzamčené.
5. Na stránce Můj přístup načtěte správcovský soubor ze soukromého balíčku a spusťte Kontrolu Studia.
6. Ochranu přímých adres doplňte do samostatných repozitářů podle `src/integration/README.md`.
