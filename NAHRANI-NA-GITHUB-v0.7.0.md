# Nahrání AI Studia GHRAB 0.7.0 na GitHub

1. Otevřete repozitář `AI-Studio-GHRAB`.
2. Zvolte **Add file → Upload files**.
3. Rozbalte dodaný ZIP a nahrajte **obsah jeho kořene**, nikoli nadřazenou složku.
4. Potvrďte změny do hlavní větve.
5. Počkejte na zelený běh GitHub Actions a nové nasazení Pages.
6. Otevřete AI Studio, proveďte tvrdé obnovení stránky a zkontrolujte záložku **Manuály**.

Dílčí aplikace s manuály musí být nasazené dříve než tato verze Studia.
