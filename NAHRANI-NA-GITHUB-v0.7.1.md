# Nahrání AI Studia GHRAB 0.7.1 na GitHub

1. Otevřete repozitář `AI-Studio-GHRAB`.
2. Zvolte **Add file → Upload files**.
3. Rozbalte dodaný ZIP a nahrajte **celý obsah jeho kořene**, nikoli nadřazenou složku.
4. Potvrďte přepsání stávajících souborů a změny odešlete do hlavní větve.
5. Počkejte na zelený běh GitHub Actions a nové nasazení GitHub Pages.
6. V prohlížeči otevřete AI Studio a proveďte tvrdé obnovení `Ctrl + F5`.
7. Zkontrolujte pořadí navigace: **Materiály → Manuály → Bezpečnost → Můj přístup**.

Stávající přístupové soubory ani oprávnění není nutné měnit.
