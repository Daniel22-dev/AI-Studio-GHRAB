# Nahrání AI Studia GHRAB 0.7.2 na GitHub

1. Rozbalte distribuční ZIP.
2. V repozitáři `AI-Studio-GHRAB` nahrajte celý obsah rozbalené složky přímo do kořene.
3. Potvrďte přepsání stávajících souborů.
4. Počkejte na zelený běh GitHub Actions.
5. Spuštěnou aplikaci AI Studio úplně zavřete a znovu otevřete, aby se aktivoval nový service worker.
6. V záložce **Manuály** otevřete libovolný odemčený průvodce. Musí zůstat uvnitř okna AI Studia.

Stávající podepsaná oprávnění se nemění a není potřeba je vydávat znovu.
