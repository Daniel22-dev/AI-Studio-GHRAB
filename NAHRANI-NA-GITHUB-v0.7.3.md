# Nahrání AI Studia GHRAB 0.7.3 na GitHub

1. Rozbalte distribuční ZIP.
2. Nahrajte celý obsah přímo do kořene repozitáře `AI-Studio-GHRAB`.
3. Potvrďte přepsání existujících souborů.
4. Počkejte na úspěšné dokončení GitHub Actions.
5. Na telefonu aplikaci úplně zavřete a znovu spusťte. Pokud se stará podoba drží v cache, jednou otevřete webovou adresu v prohlížeči a obnovte ji.

Verze 0.7.3 opravuje ořezávání čísel verzí na kartách aplikací v mobilním zobrazení.
