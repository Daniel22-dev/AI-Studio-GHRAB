# Nahrání AI Studio GHRAB v0.16.6 na GitHub

1. Na stránce neúspěšného nebo běžícího workflow zvol `Cancel workflow`, pokud stále běží.
2. Rozbal ZIP verze 0.16.6.
3. Nahraj celý obsah do kořene repozitáře a potvrď přepsání souborů.
4. Doporučený commit: `v0.16.6 – oprava npm registru a GitHub buildu`.
5. Nový workflow by měl krok `Install dependencies` dokončit obvykle během několika sekund až desítek sekund.
6. Po zeleném nasazení udělej v aplikaci tvrdé obnovení nebo ji úplně zavři a znovu otevři.
