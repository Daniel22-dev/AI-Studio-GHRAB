# Komentář změn 0.5.0

Verze 0.5.0 řeší kritické bezpečnostní nálezy z auditu a připravuje AI Studio na řízený pilot.

## Hlavní změny

- anonymní exporty už neobsahují pracovní prostor, názvy materiálů, prompty ani volné poznámky;
- přidán lokální demonstrační režim proškolení pro verzi bez serveru;
- na mobilu zůstává dostupný přepínač jazyka;
- sjednoceno zápatí: Autor a vývojový garant: Daniel Baláž / Školní projekt Gymnázia, Ostrava-Hrabůvka;
- doplněna diagnostika přímo v AI Studiu;
- připraven model Top 4 aplikací kolem jádra a katalog ostatních aplikací;
- upraveny statusy aplikací tak, aby nepůsobily jako plně oficiální školní serverový provoz.

## Důležité upozornění

Lokální režim proškolení je pouze pilotní pomůcka. Skutečné blokování přístupu musí v oficiálním provozu řešit server, školní přihlášení a API brána.
