# AI Studio GHRAB 0.8.0 — prémiová hvězdná brána

## Co se změnilo

- Centrální portál na úvodní stránce nyní vychází z původního vizuálního konceptu AI Studia.
- Přibyla vícevrstvá animace: rotující prstence, energetický vír, světelný sken, jiskry, odlesky a prostorová platforma.
- V plném režimu se brána velmi jemně naklání podle pohybu ukazatele.
- Úsporný a vypnutý režim animace omezuje nebo vypíná; respektuje se také systémové nastavení omezení pohybu.
- Funkce aplikací, přístupová oprávnění, Top 4, manuály a PWA zůstaly beze změny.
