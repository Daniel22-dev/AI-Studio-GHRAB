# AI Studio GHRAB 0.8.2 — oprava polohy brány

Na plném animačním režimu přepisovala animace hlavního obrázku jeho středové posunutí. Výsledkem byl samostatný obrázek brány vpravo dole, zatímco světelné kruhy zůstaly uprostřed.

Oprava zachovává středové posunutí v každém snímku animace. Stejná pojistka byla doplněna i pro svislý světelný paprsek.
