# AI Studio GHRAB 0.8.3 — pilotní souhrny a úklid rozhraní

## Co se změnilo

- Stránka **Můj přístup** nově obsahuje tlačítko pro stažení anonymního souhrnu pro správce pilotu.
- Stránka **Report** umí načíst více anonymních JSON souhrnů od kolegů a spojit je s místními daty.
- Pilot už není veden jako pevný 12týdenní plán, ale jako volné fáze školního roku.
- Samostatný tab **Pilot** byl odstraněn z horní administrátorské lišty; hlavní administrátorský rozcestník zůstává **Správa**.
- Duplicitní odkazy v patičce jsou skryté.
- Prezentační lišta už nepřekrývá úvodní text.
- Report používá nové logo/bránu.

## Jak se teď sbírají data od kolegů

1. Kolega používá AI Studio běžným způsobem.
2. AI Studio lokálně počítá spuštění aplikací a pracovní události.
3. Kolega otevře **Můj přístup** a stáhne anonymní souhrn.
4. Soubor pošle správci pilotu.
5. Správce otevře **Správa → Anonymní report** a soubory importuje.

Souhrn neobsahuje jména, prompty, texty materiálů, názvy materiálů ani volné poznámky.
