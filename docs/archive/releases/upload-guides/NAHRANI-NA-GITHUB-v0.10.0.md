# Nahrání AI Studia GHRAB 0.10.0

AI Studio nahrajte jako poslední, po úspěšném nasazení všech čtyř repozitářů dílčích aplikací.

Rozbalte distribuční ZIP a nahrajte jeho obsah přímo do kořene repozitáře AI-Studio-GHRAB, včetně skryté složky `.github`. Nechte soubory přepsat, potvrďte commit a vyčkejte na zelené Actions/Pages.

Po nasazení proveďte `Ctrl + F5`, zavřete staré karty/PWA dílčích aplikací a otevřete je znovu přes Studio.
