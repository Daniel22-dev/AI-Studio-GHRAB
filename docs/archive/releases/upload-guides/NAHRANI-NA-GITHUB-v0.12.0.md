# Nahrání AI Studio GHRAB 0.12.0 na GitHub

1. Rozbalte ZIP AI Studia.
2. Nahrajte obsah rozbalené složky přímo do kořene repozitáře AI Studio GHRAB.
3. Nechte přepsat stávající soubory a ověřte, že zůstala složka `.github/workflows/deploy.yml`.
4. Proveďte commit a vyčkejte na zelené GitHub Actions.
5. Otevřete nasazené Studio, stiskněte `Ctrl + F5`, zavřete staré karty/PWA dílčích aplikací a spusťte je znovu přes Studio.

AI Studio nahrávejte až jako poslední, po nasazení všech dílčích aplikací.
