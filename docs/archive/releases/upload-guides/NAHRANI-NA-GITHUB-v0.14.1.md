# Nahrání AI Studia GHRAB 0.14.1 na GitHub

1. Rozbalte balík.
2. Otevřete složku `AI-Studio-GHRAB-main`.
3. Nahraďte jejím obsahem soubory v kořeni repozitáře AI Studia.
4. V GitHub Desktopu vytvořte commit, například `AI Studio 0.14.1 – oprava importu evidence`.
5. Klikněte na `Push origin`.
6. Počkejte na zelené dokončení GitHub Actions.
7. Ve Studiu použijte `Ctrl + F5`; u nainstalované PWA ji jednou úplně zavřete a znovu spusťte.

Soukromý soubor pro přidání konkrétní kolegyně se na GitHub nenahrává.
