# Nahrání AI Studio GHRAB 0.14.2 na GitHub

1. Rozbalte ZIP.
2. Otevřete složku `AI-Studio-GHRAB-main`.
3. Zkopírujte celý její obsah do kořene naklonovaného repozitáře.
4. Potvrďte nahrazení souborů.
5. V GitHub Desktop vytvořte commit, například `Oprava importu platnosti přístupů`.
6. Klikněte na `Push origin`.
7. Po zeleném nasazení obnovte Studio pomocí `Ctrl + F5`; nainstalovanou PWA úplně zavřete a znovu spusťte.
8. V Evidenci znovu načtěte původní soubor kolegyně. Záznam se podle stejného JTI opraví bez duplicity.
