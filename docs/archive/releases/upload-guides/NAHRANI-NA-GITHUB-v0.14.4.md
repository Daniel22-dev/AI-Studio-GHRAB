# Nahrání AI Studio GHRAB 0.14.4 na GitHub

1. Otevřete repozitář **AI-Studio-GHRAB** na GitHubu.
2. Nahrajte celý obsah této složky do kořene repozitáře a potvrďte nahrazení existujících souborů.
3. Commit pojmenujte například: `Oprava navigace správcovských nástrojů 0.14.4`.
4. Počkejte, až GitHub Actions dokončí nasazení zelenou kontrolou.
5. V AI Studiu proveďte tvrdé obnovení stránky. U nainstalované PWA ji úplně zavřete a znovu otevřete.
6. Ověřte **Správa → Evidence přístupů** a následně **Vydat nový přístup**. Na obou stránkách musí zůstat kompletní horní menu.
