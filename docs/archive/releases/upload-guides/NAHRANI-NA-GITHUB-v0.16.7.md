# Nahrání AI Studio GHRAB v0.16.7 na GitHub

1. Rozbal balíček `AI-Studio-GHRAB-v0.16.7-GITHUB-READY.zip`.
2. Nahraj celý jeho obsah do kořene repozitáře AI Studia.
3. Potvrď přepsání existujících souborů.
4. Doporučený commit:

   `v0.16.7 – oprava formátování registru v GitHub Actions`

5. Sleduj nový běh v záložce **Actions**.

Krok `Synchronise application manifests` nyní sám naformátuje generované JSON soubory. Následující `Validate and build` proto nemá skončit na hlášce `Code style issues found in src/config/apps.generated.json`.
