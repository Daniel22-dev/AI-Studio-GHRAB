# Nahrání AI Studio GHRAB 0.8.0 na GitHub

1. Rozbalte ZIP.
2. Otevřete repozitář `AI-Studio-GHRAB`.
3. Nahrajte **veškerý obsah rozbalené složky přímo do kořene repozitáře** a potvrďte přepsání existujících souborů.
4. Nenahrávejte samotnou nadřazenou složku jako další úroveň.
5. Počkejte, až GitHub Actions doběhne zeleně.
6. Otevřete AI Studio, proveďte tvrdé obnovení stránky (`Ctrl+F5`) nebo ukončete a znovu spusťte nainstalovanou aplikaci.
7. Ověřte desktop i telefon a v nastavení postupně vyzkoušejte plný, úsporný a vypnutý režim animací.

Soukromé klíče ani vydané přístupové soubory se na GitHub nenahrávají.
