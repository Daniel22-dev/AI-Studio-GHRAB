# Nahrání AI Studio GHRAB v0.17.0 na GitHub

1. Rozbal ZIP.
2. Nahraj celý obsah přímo do kořene repozitáře AI Studia.
3. Potvrď přepsání existujících souborů.
4. Doporučený commit: `v0.17.0 – čistý základ, dvoufázové spuštění a instalace na PC`.
5. Po zeleném dokončení GitHub Actions proveď `Ctrl + F5` nebo úplně ukonči a znovu otevři nainstalovanou PWA.

## Kontrola po nasazení

- vpravo dole je na PC instalační karta;
- po kliknutí na aplikaci se nejprve 2 sekundy otáčejí prstence hlavní brány;
- poté se zobrazí samostatná animace konkrétní aplikace;
- po animaci se aplikace otevře.
