# Nahrání AI Studio GHRAB v0.17.1 na GitHub

1. Rozbalte `AI-Studio-GHRAB-v0.17.1-GITHUB-READY.zip`.
2. Nahrajte celý obsah rozbalené složky do kořene repozitáře AI Studia.
3. Potvrďte přepsání existujících souborů.
4. Doporučený commit:

   `v0.17.1 – aplikace uvnitř Studia a oprava bílého popupu`

5. Po zeleném dokončení GitHub Actions proveďte `Ctrl + F5`.
6. U nainstalované PWA ji úplně zavřete a znovu otevřete.

## Rychlá kontrola

- kliknutí nevytvoří bílou záložku;
- dvě sekundy jsou vidět otáčející se prstence;
- následně se zobrazí animace aplikace;
- aplikace se otevře uvnitř okna AI Studia;
- tlačítko **AI Studio** vlevo nahoře vrátí uživatele zpět;
- karta **Nainstalovat AI Studio** se na podporovaném desktopovém prohlížeči zobrazuje vpravo dole, pokud Studio ještě není nainstalováno.
