# Ověření integrace zámku

Pro každou aplikaci proveďte:

1. Anonymní prohlížeč, přímá URL aplikace: zobrazí se zámek.
2. Oprávnění pro jinou aplikaci: stále zámek.
3. Správné učitelské oprávnění: aplikace se spustí.
4. Správcovské oprávnění: aplikace se spustí.
5. Expirované nebo zneplatněné oprávnění: zámek s přesným vysvětlením.
6. Offline režim: použije se poslední dostupný ověřovací seznam; po návratu online se ověří aktuální seznam zneplatnění.
