Společné postupy celého ekosystému

Tato příručka doplňuje manuály jednotlivých aplikací. Popisuje témata, která se mají provádět stejně v GITu, Diferenciátoru, Hodnotiteli, Korespondenčním asistentovi, LUDUSu i ACTIVA.

API klíče a modely

  1. API klíč vkládejte pouze do nastavení konkrétní aplikace.
  2. Klíč neposílejte e-mailem a nevkládejte jej do projektu, screenshotu ani diagnostického protokolu.
  3. Rychlý model používejte pro běžnou tvorbu, důkladný režim pro náročné podklady.
  4. Při chybě limitu zkontrolujte stav klíče a až potom opakujte požadavek.
AI výstup vždy kontroluje učitel. Připojení modelu nenahrazuje odbornou ani pedagogickou odpovědnost.

Soukromí a anonymizace

Nevkládejte celá jména žáků, osobní e-maily, adresy, rodná čísla, zdravotní údaje ani citlivé rodinné informace. Používejte anonymní označení typu ŽÁK_01. Kontrola aplikace je pomocná; konečnou kontrolu provádí učitel.

PDF, tisk a náhled

Před exportem zkontrolujte řešení, dělení stran, velikost písma, variantu a režim žák/učitel. Pro běžné kopírování volte úsporný černobílý režim. Systémové „Uložit jako PDF“ zachová přesnou sazbu náhledu.

Lokální data, zálohy a obnova

Bezserverové aplikace ukládají část dat do konkrétního prohlížeče. Důležitý materiál proto exportujte jako projekt nebo knihovní balíček. Vymazání dat prohlížeče může místní projekty odstranit. API klíče do záloh nepatří.

Instalace PWA a aktualizace

AI Studio lze v Chrome nebo Edge nainstalovat jako aplikaci. Po vydání nové verze zavřete stará okna a znovu spusťte Studio. Pokud se zobrazuje stará verze, použijte obnovení aplikace v pracovním prostoru nebo vyčkejte na dokončení aktualizace offline cache.

Testy, diagnostika a hlášení chyby

  1. Zopakujte chybu a poznamenejte přesný krok.
  2. Otevřete interní testy nebo diagnostiku aplikace.
  3. Zkontrolujte a anonymizujte snímek obrazovky.
  4. Přiložte verzi aplikace, technický protokol a stručný popis očekávaného chování.
Nefotografujte monitor telefonem. Použijte snímek obrazovky a před odesláním začerněte citlivé údaje.
